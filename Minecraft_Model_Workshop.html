ระบบ Coordinate ใน Bedrock

Y (บน) | | / Z (หน้าหลัง) | / | / +---------- X (ซ้ายขวา) จุดกลาง [0, 0, 0] = ตรงกลางพื้น ผู้เล่นยืน: เท้า = Y 0, หัว = Y 28 1 block = 16 × 16 × 16 unit

โครงสร้าง geo.json

geometry.my_mob ← ชื่อ model │ └── bone: root [pivot 0,0,0] ← จุดยึดหลัก │ ├── bone: body [pivot 0,12,0] │ └── cube origin[-4,6,-2] size[8,12,4] │ └── bone: head [pivot 0,24,0] └── cube origin[-4,20,-4] size[8,8,8]
💡 pivot คืออะไร pivot = จุดหมุนของ bone ถ้า head มี pivot ที่คอ (Y=24) มันจะหมุนรอบคอ = ถูกต้อง ถ้า pivot อยู่กลางหัว = หมุนผิดท่า

root [0, 0, 0] blade_main origin[-0.5, 4, -0.5] size[1, 14, 1] ← ใบดาบหลัก blade_edge_L origin[-1, 6, -0.25] size[0.5,10,0.5] ← ขอบซ้าย blade_edge_R origin[0.5, 6, -0.25] size[0.5,10,0.5] ← ขอบขวา blade_tip origin[-0.25,18,-0.25] size[0.5, 2,0.5] ← ปลาย crossguard origin[-2.5, 3, -0.5] size[5, 1, 1] ← กาดมือ handle origin[-0.5, 0, -0.5] size[1, 3, 1] ← ด้ามจับ pommel origin[-1, -1, -1] size[2, 1, 2] ← หัวด้าม

ขั้นตอนสร้าง Item Model

RP/attachables/crystal_blade.json

{
  "format_version": "1.10.0",
  "minecraft:attachable": {
    "description": {
      "identifier": "myaddon:crystal_blade",
      "materials": {
        "default": "entity_alphatest",
        "enchanted": "entity_alphatest_glint"
      },
      "textures": {
        "default": "textures/items/crystal_blade",
        "enchanted": "textures/misc/enchanted_item_glint"
      },
      "geometry": { "default": "geometry.crystal_blade" },
      "render_controllers": ["controller.render.item_default"]
    }
  }
}
📌 Item 3D ต้องมี 2 ไฟล์ 1) geo.json — รูปร่างดาบ   2) attachable.json — บอกว่าให้แสดงในมือผู้เล่น ถ้าไม่มี attachable.json เกมจะแสดงแค่ sprite 2D ธรรมดา

{
  "format_version": "1.21.0",
  "minecraft:block": {
    "description": {
      "identifier": "myaddon:crystal_pillar",
      "menu_category": { "category": "nature" }
    },
    "components": {
      "minecraft:geometry": {
        "identifier": "geometry.crystal_pillar"
      },
      "minecraft:material_instances": {
        "*": {
          "texture": "crystal_pillar",
          "render_method": "alpha_test"
        }
      },
      "minecraft:collision_box": {
        "origin": [-4, 0, -4],
        "size": [8, 16, 8]
      },
      "minecraft:light_emission": { "emission": 3 },
      "minecraft:destructible_by_mining": { "seconds_to_destroy": 1.5 }
    }
  }
}
⚠️ collision_box ต้องกำหนดเอง ถ้าไม่กำหนด Minecraft ใช้ full cube แทน — ผู้เล่นชนกับอากาศรอบๆ บล็อค

        "idle": {
          "animations": ["idle"],

ขั้นตอน Import

ปัญหาที่เจอบ่อย

ปัญหาสาเหตุวิธีแก้
Model ไม่โชว์identifier ไม่ตรงกันเช็ค "geometry.xxx" ให้เหมือนทุกไฟล์
Texture สีม่วงpath ผิดเช็ค path ใน entity.json
Armor ทะลุตัวcube เล็กเกินเพิ่ม size +0.5 unit
Animation ไม่เล่นลืม scripts.animateเพิ่ม "animate": ["ctrl"]
Block ชนอากาศไม่มี collision_boxเพิ่ม minecraft:collision_box
Item ไม่โชว์ใน handไม่มี attachable.jsonสร้าง RP/attachables/item.json
